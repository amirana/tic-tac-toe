# Tic Tac Toe

A classic two-player Tic Tac Toe game built in React. Players take turns marking cells on a 3×3 grid. The first player to align three marks horizontally, vertically, or diagonally wins.

## Demo

[Live Demo](https://amirana.me) 

## Features

- Two-player gameplay (Player 1 vs Player 2)
- Win detection across all rows, columns, and diagonals
- Draw detection when the board is full
- Turn indicator showing the current player
- Restart button to reset the game

## Tech Stack

- React
- CSS Grid

## Getting Started

### Prerequisites

- Node.js v18+
- pnpm

### Installation

```bash
git clone https://github.com/amirana/tic-tac-toe
cd tic-tac-toe
pnpm install
pnpm run dev
```

Open `http://localhost:5173` in your browser.
